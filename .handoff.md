   (full cut → identical draw, healthy → none).
2. **Graded extraction proxy (commit `f162559`).** Per-sector trailing EWMA output
   norm (`sectorOutputNorm`, serialized). `sectorRawLevel` grades off output/norm:
   ≥ norm → 1.0 (healthy play byte-clean); a contraction below the deadband (0.9)
   grades toward MIN_LEVEL (0.35) at the floor (0.5). Norm chases output → the drag
   bites the *transition* into a downturn, then heals. Damped on three axes (floor
   ≠ 0, 1-month-lagged mult, 15% max drag) → non-divergent.

## 4. FAILED PATHS / GOTCHAS (save the next agent time)

- **Byte-identity hinges on the deadband + the EWMA chasing growth.** In steady
  or growing play output ≥ its trailing norm → `sectorRawLevel` clamps to 1.0 →
  no drag → the suite is unperturbed. The whole 906-test suite passes *with* phase
  2 live precisely because no healthy-play test dips output below the 0.9 deadband.
  If you tighten the deadband or steepen the curve, expect to re-baseline.
- **The graded proxy IS an intentional balance change** — but it's invisible until
  a real contraction (recession/disaster/war), confined to the industry drag
  (≤15%), and self-healing (the norm re-baselines). Verify with the headless sim
  (`npx tsx src/sim/headless.ts 181 8`), NOT just unit tests — a doom-loop would
  show as a seed spiralling to ~0 GDP/pop. None did; large economies barely move.
- **The goods are STILL an abstract layer.** Stocks feed only `supplyChainHealth`
  (→ drag) + the two secondary effects (pharma→plague, electronics→research).
  Nothing reads goods/levels into GDP, prices, or trade. The graded plumbing is
  groundwork for that, not the payoff — wiring real consumption/prices is next.
- **`supplyShockSeverity` reads the *cached* `supplyChainHealth`**, not the live
  snapshot health. To assert a drag in a test you must `tickIntermediateGoods()`
  first; to assert a raw *level* without a tick, use `supplyChainSnapshot()` (pure
  read — it evaluates `sectorRawLevel` but does NOT advance the norm).
- **`npm install`:** `ELECTRON_SKIP_BINARY_DOWNLOAD=1 PLAYWRIGHT_SKIP_BROWSER_DOWNLOAD=1
  npm install --ignore-scripts --no-audit --no-fund` (electron's postinstall 403s
  in this web env and rolls back the whole install).
- **Asset/audio generation** still network-blocked (`huggingface.co` 403, no
  sharp/ffmpeg). Pipeline plumbing is done + dry-run-verified; generate locally.
