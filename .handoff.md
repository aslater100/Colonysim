## 4. FAILED PATHS / GOTCHAS (save the next agent time)

- **The one bug the golden diff caught, don't reintroduce it:** compute the
  supply-shock mult in `tickIntermediateGoods` (where `supplyChainHealth` and
  `year` are the same month), **not** later in `updateSectors`. Computed at the
  next `updateSectors`, a January year-roll compares last month's health against
  *this* year's structural baseline and fabricates a phantom shock at every era
  boundary — which silently perturbed treasury (GDP stayed identical, so it hid in
  a non-obvious field). A golden before/after snapshot across seeds is the way to
  catch this class of lag bug — unit tests alone missed it.
- **The drag is binary today:** the raw proxy treats coal/iron/copper as available
  iff *total* industry output > 0, so the drag only fires on a full industry
  shutdown (deep depression / war), modelling catastrophe, not friction. That's by
  design but limits how often it matters — a graded per-raw proxy is the upgrade.
- **`npm install` fails on a fresh container** because `electron@33.2.0`'s
  postinstall downloads its binary and this web env's egress blocks it (the whole
  install rolls back → empty `node_modules`). Fix used: `ELECTRON_SKIP_BINARY_DOWNLOAD=1
  PLAYWRIGHT_SKIP_BROWSER_DOWNLOAD=1 npm install --ignore-scripts --no-audit --no-fund`.
  CI's `test.yml` already sets `ELECTRON_SKIP_BINARY_DOWNLOAD=1`.
- **Asset/audio generation (Track A / B2-audio) is still network-blocked** —
  `huggingface.co` egress 403s, no `sharp`/`ffmpeg` encoder here. Pipeline plumbing
  (`assetCatalog.ts`, `hf-assets.ts`, `audioRegistry.ts`) is done + dry-run-verified.
  Generate locally or re-provision egress.
- **Determinism is the load-bearing constraint.** `region.ts` is a ~13.9k-line flat
  `serialize()`/`?? default` backfill; any new field must serialize + backfill, and
  any RNG-consuming change must keep the same consumption order. `region-longrun`
  (fixed-seed 50-yr run) is the guard. For no-RNG economic multipliers, also run a
  golden before/after snapshot — the determinism test is same-seed self-consistency,
  it won't catch a real behaviour shift.
