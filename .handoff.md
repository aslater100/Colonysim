## 4. FAILED PATHS / GOTCHAS (save the next agent time)

- **`npm install` fails on a fresh container** because `electron@33.2.0`'s
  postinstall downloads its binary and this web env's egress blocks it (the
  whole install then rolls back → empty `node_modules`). Fix used:
  `ELECTRON_SKIP_BINARY_DOWNLOAD=1 PLAYWRIGHT_SKIP_BROWSER_DOWNLOAD=1 npm install
  --ignore-scripts --no-audit --no-fund`. CI's `test.yml` already sets
  `ELECTRON_SKIP_BINARY_DOWNLOAD=1`, so CI installs fine without `--ignore-scripts`.
- **Asset/audio generation (Track A / B2-audio) is still network-blocked** —
  `huggingface.co` egress 403s and there's no `sharp`/`ffmpeg` encoder here. Do
  NOT attempt asset *generation* from this web env; the pipeline plumbing
  (`assetCatalog.ts`, `hf-assets.ts`, `audioRegistry.ts`) is already done and
  dry-run-verified. Generate locally or re-provision egress. (Confirmed again
  this session — unchanged.)
- **Don't naively wire `supplyChainHealth` into output.** It's a dead metric
  today (computed + serialized, never consumed by the economy). Tempting to
  multiply industrial output by it, but health legitimately dips below 1.0 in
  *healthy* play at each good's era-unlock boundary (e.g. vehicles unlocks 1925
  but needs components which unlock 1930 → health 0.5 across 1925–30), so a naive
  multiplier perturbs the early-game balance and the economy-balance/long-run
  integration tests. Baseline or gate it if you wire it.
- **Determinism is the load-bearing constraint.** `region.ts` is a 13.8k-line
  flat `serialize()`/`?? default` backfill; any new field must serialize +
  backfill, and any change to RNG-consuming code must keep the same consumption
  order. The `region-longrun` test (deterministic over a fixed-seed 50-yr run)
  is the guard — run it after any sim-core change.
- **Phase status:** the `HANDOFF.md` roadmap prose lags the code — Phases 1–18,
  FX, parallax backdrop, audio registries, and zoning/utilities (`tickUtilities`,
  `computeLandValue`) are **already landed**, despite some sections reading
  "unimplemented". Grep the code before assuming a system is a greenfield gap.
